**数据库（database）保存有组织的数据的容器（通常是一个文件或一组文件）**

那什么又是SQL呢？先来看一下全称，Structured Query Language，结构化查询语言，既然是语言，那么就像是其他大部分语言一样，有它的关键字，语法等等，以及它主要是用于做什么，就像HTML，CSS，JavaScript用来做网站的前端，PHP，JAVA等通常用于后端，SQL则是用于和数据库交互通信
